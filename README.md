## 📦 رفع التطبيق على GitHub Pages

### الخطوة 1 — إنشاء حساب GitHub
1. ادخل إلى **github.com**
2. اضغط **Sign up** وأنشئ حساباً (مجاني)
3. أكّد بريدك الإلكتروني

### الخطوة 2 — إنشاء مستودع جديد
1. اضغط على **+** في الأعلى → **New repository**
2. **Repository name:** `smart-eps` (أو أي اسم تفضله)
3. اختر **Public** ✅
4. اضغط **Create repository**

### الخطوة 3 — رفع ملفات التطبيق
1. في صفحة المستودع، اضغط **uploading an existing file**
2. اسحب **كل الملفات** من مجلد `smart_eps_pwa` دفعة واحدة:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `favicon.png`
   - كل ملفات `icon-*.png`
3. أسفل الصفحة اكتب رسالة: `Initial upload`
4. اضغط **Commit changes**

### الخطوة 4 — تفعيل GitHub Pages
1. في المستودع، اضغط **Settings** (الإعدادات)
2. من القائمة اليسرى اختر **Pages**
3. تحت **Source** اختر:
   - Branch: **main**
   - Folder: **/ (root)**
4. اضغط **Save**
5. انتظر دقيقة أو دقيقتين

سيظهر رابط مثل:
```
https://اسم-حسابك.github.io/smart-eps/
```

✅ هذا هو رابط تطبيقك على الإنترنت!

## 📱 تثبيت التطبيق على الهاتف

### على أندرويد (Chrome)
1. افتح رابط GitHub Pages في **Chrome**
2. اضغط على القائمة (3 نقاط في الأعلى)
3. اختر **Install app** أو **Add to Home screen**
4. ستظهر أيقونة Smart EPS في الشاشة الرئيسية كتطبيق مستقل

### على iPhone (Safari)
1. افتح الرابط في **Safari**
2. اضغط زر **المشاركة** (المربع مع السهم)
3. اختر **Add to Home Screen** (إضافة إلى الشاشة الرئيسية)
4. اضغط **Add**

### مميزات بعد التثبيت
- ✅ يفتح كتطبيق مستقل (بدون شريط المتصفح)
- ✅ يعمل **أوفلاين** بعد أول فتح
- ✅ أيقونة Smart EPS على الشاشة الرئيسية
- ✅ سرعة فتح فورية

## 🔄 تحديث التطبيق لاحقاً

عندما تريد تحديث الملف:
1. ادخل إلى مستودع GitHub
2. اضغط على ملف `index.html`
3. اضغط أيقونة القلم (تعديل)
4. الصق محتوى الملف الجديد أو ارفع نسخة جديدة
5. اضغط **Commit changes**
6. التطبيق سيتحدث تلقائياً للمستخدمين خلال دقائق

---

## ⚠️ ملاحظات مهمة

- **يجب أن يكون الرابط HTTPS** — GitHub Pages يوفر هذا تلقائياً
- **الميكروفون والكاميرا** لا تعمل إلا على HTTPS
- البيانات تُحفظ محلياً على جهازك (LocalStorage)
- استخدم زر **النسخة الاحتياطية JSON** بشكل دوري
